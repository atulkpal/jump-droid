## 1. Executive Summary
v1.5.2 marks the transition to the **Closed Beta** phase. This release focuses on UX improvements (Continue-Ready overlay, Sharing system), AI refinement (Heat Bat behavioral states), and long-term engagement features ("NEW" badge system for the Archive).

## 2. Key Features
- **Continue-Ready Overlay**: Adds a "Tap to resume" buffer after continues to prevent accidental deaths.
- **Sharing System**: Native Android share intent integration for sharing altitude scores.
- **Archive "NEW" Badges**: Visual indicators for unread Codex discoveries.
- **Heat Bat AI**: 4-state behavioral engine (Searching, Attacking, Orbiting, Fleeing).
- **Boss Cooldowns**: 60s global cooldown between boss spawns for better pacing.
- **Main Menu Responsiveness**: Improved layout scaling for varied screen sizes.
