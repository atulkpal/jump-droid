## Release Summary

This is a production hotfix release addressing two identical crash patterns in the audio system:

1. **Portrait lock** — App-wide `android:screenOrientation="portrait"` prevents landscape rotation across all screens.
2. **Keep screen on** — `FLAG_KEEP_SCREEN_ON` is active during gameplay (PLAYING, ASCENSION_PROTOCOL, PAUSED) and cleared on game over, navigation away, or app backgrounding.

### Bug Fixes

3. **Crash on game over** — `SoundManager.fadeOutAndPlayGameOver()` threw `IllegalStateException` from `MediaPlayer.setVolume()` when the player was already released by a concurrent operation. The volume fade loop and player cleanup are now wrapped in try-catch for defensive safety. (Fixes crash session `6A54716401E50001768F4B070BE69903`.)
4. **Crash during music crossfade** — `SoundManager.playMusic()` threw the same `IllegalStateException` in the crossfade coroutine when rapid zone changes triggered concurrent release of either the old or new player. The entire crossfade body is now wrapped in try-catch. (Fixes crash session `6A551A8100B200017167E85F01DE29A5`.)

No gameplay, architecture, or release pipeline changes. All EPIC 11 content preserved.
