## About Jump Droid

Jump Droid is a precision vertical exploration simulator where players pilot a droid-controlled rocket through hostile atmospheric zones. The game features 12 unique zones, 11 boss encounters, modular rocket customization, a 12-track mission system, and deep endgame content including prestige and Eternal Mode.

Built entirely with Kotlin and Jetpack Compose Canvas, Jump Droid is open source under the MIT License. The complete source code is available on GitHub.
