## Screenshot Strategy

Order screenshots to tell a story:

1. **Hero** — Title screen or atmospheric zone establishing shot
2. **Gameplay** — Core gameplay with HUD visible
3. **Boss fight** — Dramatic boss encounter moment
4. **Customization** — Hangar / build screen
5. **Progression** — Mission or zone discovery card
6. **Endgame** — Ascension Ceremony or Eternal Mode

Each screenshot should include a 1-line overlay caption (e.g., "Face 11 unique bosses" or "Build your perfect rocket").

## Open Source Advantage

Jump Droid is fully open source (MIT). Developers can inspect, learn from, and fork the codebase. This differentiates it from nearly all other mobile games and attracts a technical audience. Highlight "Built with Kotlin and Jetpack Compose" prominently.
