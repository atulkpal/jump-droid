# 🚀 Jump Droid

A vertical sci-fi exploration game where a lone rocket climbs from Earth through the clouds, atmosphere, deep space, and beyond.

Jump Droid began as a simple platform-jumping prototype and is evolving into a complete exploration adventure featuring discoveries, progression systems, atmospheric worlds, bosses, achievements, and unlockable content.

## Gameplay

Pilot a rocket through an ever-changing vertical world.

Manage:

* Fuel
* Heat
* Momentum
* Platform selection
* Powerup collection

Climb through multiple altitude zones while discovering secrets and unlocking Codex entries.

---

## Current Features

### 🚀 Rocket Classes

Multiple playable rocket types with unique handling characteristics.

Examples:

* Scout
* Tank
* Additional classes planned

---

### 🌎 World & Atmosphere

Travel through multiple altitude zones:

* Earth
* Cloud Layer
* Upper Atmosphere
* Space
* Deep Space
* Void

Each zone introduces new visuals, ambience, and discoveries.

---

### 📖 Discovery & Codex System

As players explore, they unlock discoveries that populate the in-game Codex.

The Codex serves as a record of the player's journey through the universe.

---

* Temporary boosts
* Utility powerups
* Rare artifacts

Additional powerups are planned for future releases.

## Project Philosophy

Jump Droid is being developed incrementally through large feature-driven EPICs.

Each EPIC is:

* Designed
* Implemented
* Playtested
* Tagged
* Released

This allows the complete evolution of the game to be tracked through Git history and GitHub Releases.
