### Common Issues

| Issue | Solution |
|-------|----------|
| Game crashes on launch | Ensure Android 14+ (API 34). Update to latest version. |
| Audio not playing | Check device volume. Toggle mute in Settings. |
| Game performance issues | Close background apps. Reduce device brightness. |
| Can't build from source | See [Installation section](README.md#-installation) in README. |
